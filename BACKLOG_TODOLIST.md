## User Stories

As a user, I want to add tasks, so that I can manage my work.

As a user, I want to remove tasks, so that I can delete tasks I no longer need.

As a user, I want to mark tasks as complete, so that I can track progress.

As a user, I want to edit tasks, so that I can correct or update task information.

As a user, I want to view all tasks, so that I can see what I need to do.

As a user, I want to filter tasks, so that I can focus on completed or incomplete tasks.
