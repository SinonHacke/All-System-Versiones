## User Stories

### US-1: User Registration
As a user, I want to create an account so that I can manage my tasks.

### US-2: User Login
As a user, I want to log in so that I can access my tasks.

### US-3: Add Task
As a user, I want to add a new task to my todo list.

### US-4: View Tasks
As a user, I want to see all my tasks with their status.

### US-5: Mark Task as Done
As a user, I want to mark tasks as completed.

### US-6: Delete Task
As a user, I want to delete tasks I no longer need.

### US-7: Data Persistence
As a user, I want my tasks to be saved so they are not lost after closing the app.
